# xctestrunner-bash

Bash port of [google/xctestrunner](https://github.com/google/xctestrunner), aiming
to be a drop-in replacement so that it supports cross-platform use.

This is all AI-agent-generated code, led by a human. The code shouldn't be
too complicated, since it's just a thin wrapper around simctl, xcdevice, and
several other tools.
